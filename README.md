# The Legend of Shray

## Installation Dependencies
* python 3
* pygame

## How to Run
```
git clone https://github.com/benson-w/The-Legend-of-Shray.git
cd The-Legend-of-Shray
python3 shray.py
```

## Controls
* WASD for movement (or arrow keys)
* esc to exit game
* TAB to respawn
* 1 to toggle gun, 2 3 4 for other weapons
* r to reload
* mouse to shoot
